PROBLEM 3: Storage and key presses
  • Input box id: my-name-is

Subscribe to the "keyup" event of the input box. When the event occurs, store the value of the input box in local storage. When the page loads, if there's a stored value in local storage for the input box, set the value of the input box to the stored value. Write code in the "your-code.js" file to make that happen.

What this means is that if someone types "Sugar face" into the text box and refreshes the page, then the text "Sugar face" should show back up in the text box after the page loads.

To get the value of what is in the text box, get the element by its id and use the value property on it to get and set the value of the input box. For example, if you had the following HTML: 

<input type="text" id="nickname">
the following code would set the value of the input box to "Dima". 
document
  .getElementById('nickname')
  .value = 'Dima';
